1 Project overview

1.1 Introduction

This book documents an inventory of environmental and biological data in the Gulf of Maine, including sources and availability. The inventory includes information about the projects, such as the lead entity, partner entities, project description, data start and end dates, project website, data links, metadata links, principal investigator (PI) name and contact information, spatial scale, data categories, data availability, data type, and notes.

The data are stored in a data frame called gom_data, which is available in this R package. The data frame contains 236 rows and 17 variables. Each row represents a project, and the variables provide information about the project and its associated data.

A full written report summarizing this data, as well as data gaps, will be available with the BOEM Environmental Study series on the GovInfo website.
